# Palette's Journal

## 2026-02-07 - [Actionable Empty States]
**Learning:** Empty states without a clear call-to-action (CTA) leave users stranded, especially in new onboarding flows. Adding a direct link or button significantly improves the user journey.
**Action:** Always include a primary CTA button in empty states that guides the user to the next logical step (e.g., "Connect Account", "Create New", "Import").
